- This application uses flask and the goodlmaps python API to take in two locations and return directions from those locations. Additionally, it will display these two location on the map with markers.
- Enter in your start and end address to see them on the map.
- The map will center on your start location. Zoom out to see the pin on the destination
- directions will be available from your start location to end location by clicking the "Directions" link at the bottom of the map page. (alternatively, you may append "/d" at the end of the URL)
